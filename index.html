Your Tidbyt, your rules.

Hubyt is a local server that runs on Windows and drives your Tidbyt display with custom cards — sports scores, weather, countdowns, and more — with no cloud required.

What it does

🖥

Runs locally

A lightweight Python server on your Windows machine. No cloud subscription, no accounts.

🃏

Card system

Each card is a single Python file. Browse and install community cards directly from the UI.

📺

Multi-device

Manage multiple Tidbyt displays from one control panel. Each gets its own card playlist.

🔆

Per-device controls

Brightness slider, quiet hours, and dwell time per card — all configurable from the browser.

⚡

Live scores

MLB, NBA, NHL, and NFL scorecards pull live data from ESPN with smart polling — fast when live, slow when not.

🔌

Open add-ons

Community cards live on GitHub. Drop a .py file in the addons folder and it loads instantly.
